# n8n Automation Projects

This repository contains structured and reusable n8n workflows designed to automate real-world business use cases including supply chain monitoring, vendor delay analysis, Telegram notifications, and chatbot integrations.

## 🚀 Use Cases Included

- Vendor Shipment Delay Analysis (Excel → JSON → JS Logic → Telegram Alert)
- Telegram Bot Automation & Alert System
- Data Transformation Workflows
- Conditional Logic & Loop-Based Automations
- Chatbot Experience Automation

## 🛠 Tech Stack

- n8n (Workflow Automation)
- JavaScript Function Nodes
- Telegram Bot API
- Excel to JSON Processing
- Conditional & Loop Logic

## 🎯 Purpose

This repository serves as:
- A portfolio of automation use cases
- A reusable workflow library
- A version-controlled backup of n8n projects
